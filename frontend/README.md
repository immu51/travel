# Frontend – TraverraX

React + Vite + Tailwind. Tour & Travel Agency website.

## Setup

```bash
npm install
cp .env.example .env   # edit .env with VITE_API_URL, Formspree, Firebase if needed
npm run dev
```

Open http://localhost:5173.

## Scripts

- `npm run dev` – Dev server
- `npm run build` – Production build → `dist/`
- `npm run preview` – Preview production build

## Env

See root **ENV-SETUP.md**. Key: `VITE_API_URL=http://localhost:5000` to use the backend API.
